# 🌙 Dark Mode Toggle Chrome Extension

A simple and lightweight Chrome Extension that allows users to switch any website between **Dark Mode** and **Light Mode** with a single click.

## ✨ Features

- 🌙 One-click Dark Mode
- ☀️ One-click Light Mode
- ⚡ Fast and lightweight
- 🎨 Clean and modern UI
- 🔧 Built with Manifest V3
- 💻 Works on most websites

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chrome Extension API
- Manifest V3

## 🚀 Installation

1. Clone this repository

```bash
git clone https://github.com/yourusername/dark-mode-toggle-extension.git
```

2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project folder

6. Start using the extension.
